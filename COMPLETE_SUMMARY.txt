════════════════════════════════════════════════════════════════════════════════
                             QUICK START (5 MINUTES)
════════════════════════════════════════════════════════════════════════════════

Step 1: Get Free Hugging Face Token
  → https://huggingface.co/settings/tokens
  → Create new read-only token
  → Copy token (format: hf_xxx...)

Step 2: Configure Environment
  → Edit: backend/.env
  → Add: HF_TOKEN=hf_your-token-here
  → Save file

Step 3: Install Dependencies
  $ cd backend
  $ pip install -r requirements.txt

Step 4: Start Backend
  $ python -m uvicorn app.main:app --reload
  ✓ Should show: "Loading HuggingFace model..."

Step 5: Start Frontend (New Terminal)
  $ cd frontend
  $ npm run dev
  ✓ Should show: "ready on 0.0.0.0:3000"

Step 6: Open Browser
  → http://localhost:3000

Step 7: Test
  1. Select department from dropdown
  2. Upload a document
  3. Ask a question
  4. See answer with sources ✅
